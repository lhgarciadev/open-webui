# Fase 0.1 - Ejecutar Prerrequisitos

## Objetivo
Configurar cuenta de Hugging Face y herramientas necesarias para la migración.

## Contexto
- Hugging Face ofrece ZeroGPU con GPU H200 gratuita
- Se requiere cuenta HF y token con permisos de escritura
- La CLI de HF facilita el deployment

## Prompt de Ejecución

```
Configura los prerrequisitos para migrar Ollama a Hugging Face:

1. CREAR/VERIFICAR CUENTA HUGGING FACE:

   a) Ir a https://huggingface.co/join
   b) Crear cuenta o iniciar sesión
   c) Verificar email si es cuenta nueva

2. CREAR TOKEN DE ACCESO:

   a) Ir a https://huggingface.co/settings/tokens
   b) Click en "Create new token"
   c) Configurar:
      - Name: cognitia-ollama
      - Type: Write (requerido para crear Spaces)
   d) Copiar token (empieza con hf_)
   e) Guardar en lugar seguro

3. INSTALAR HUGGING FACE CLI:

   # Con pip (recomendado)
   pip install -U huggingface_hub

   # Verificar instalación
   huggingface-cli --version
   # Debe mostrar versión >= 0.20.0

4. LOGIN CON CLI:

   # Iniciar sesión
   huggingface-cli login

   # Cuando pregunte por token, pegar el token creado
   # Seleccionar "Add token as git credential" = Yes

5. VERIFICAR LOGIN:

   # Confirmar usuario
   huggingface-cli whoami

   # Debe mostrar tu nombre de usuario
   # Ejemplo: "Your name: juanquiroga"

6. VERIFICAR GIT:

   # Git es necesario para push al Space
   git --version
   # Debe ser >= 2.0

   # Configurar git si no está hecho
   git config --global user.name "Tu Nombre"
   git config --global user.email "tu@email.com"

7. DOCUMENTAR CONFIGURACIÓN:

   # Guardar tu username de HF
   export HF_USERNAME="tu_usuario_hf"

   # El Space se llamará:
   # https://huggingface.co/spaces/${HF_USERNAME}/cognitia-ollama

Entrega:
- Username de Hugging Face
- Confirmación de huggingface-cli whoami
- Versión de huggingface_hub instalada
```

## Comandos Rápidos

```bash
# Instalación completa en una línea
pip install -U huggingface_hub && huggingface-cli login

# Verificar todo
huggingface-cli whoami && git --version
```

## Troubleshooting

### CLI no encuentra comando
```bash
# Verificar que pip instaló en PATH
python -m huggingface_hub.cli whoami

# Si funciona, agregar a PATH o usar alias
alias huggingface-cli="python -m huggingface_hub.cli"
```

### Login falla
```bash
# Verificar token manualmente
curl -H "Authorization: Bearer hf_xxx" \
     https://huggingface.co/api/whoami

# Si funciona, reintentar login
huggingface-cli logout
huggingface-cli login
```

### Token sin permisos
```bash
# Crear nuevo token en:
# https://huggingface.co/settings/tokens
# Asegurar Type = Write
```
