# Etapa 2.2 - Validar Mejoras de Diseño

## Objetivo

Verificar que las mejoras de diseño se aplican correctamente en las presentaciones.

## Prompt de Validación

```
Valida las mejoras de diseño:

1. GRADIENTES:
   - Generar presentación de prueba
   - Abrir en PowerPoint/LibreOffice
   - Verificar que slide de título tiene gradiente
   - Verificar que stats tiene fondo gradiente

2. TIPOGRAFÍA:
   - Verificar que títulos usan Calibri Light 44pt
   - Verificar que headings usan Calibri Bold 32pt
   - Verificar que body usa Calibri 18pt
   - Verificar colores de la paleta brand

3. SMART BULLETS:
   - Generar presentación con bullets que contengan:
     - "Tarea completada" → debe mostrar ✓
     - "Siguiente paso" → debe mostrar →
     - "Punto importante" → debe mostrar ★
     - "Información general" → debe mostrar •

4. FOOTER:
   - Verificar que slides 2+ tienen footer
   - Verificar formato "1/N" a la izquierda
   - Verificar "Powered by Cognitia" a la derecha
   - Verificar que slide de título NO tiene footer

5. ELEMENTOS DECORATIVOS:
   - Verificar círculos en slide de título
   - Verificar que no obstruyen contenido

6. COMPARACIÓN VISUAL:
   - Generar presentación antes (sin mejoras)
   - Generar presentación después (con mejoras)
   - Comparar lado a lado

Entrega:
- PASS/FAIL por cada aspecto
- Screenshots comparativos
- Archivo PPTX de prueba
```
