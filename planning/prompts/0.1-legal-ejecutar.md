# Etapa 0.1 - Ejecutar Diagnostico

## Prompt

Audita el flujo de tools y presentaciones en este repo. Identifica por qué "Presentations" no ejecuta cuando el usuario lo activa. Revisa backend y frontend para:

1. Confirmar condición de inyección de builtin tools.
2. Confirmar dependencia de `features.presentations` y `function_calling`.
3. Listar archivos exactos y líneas relevantes.
4. Proponer plan de fix mínimo y seguro.

Entrega:

- Hallazgos críticos primero.
- Riesgos de regresión.
- Plan de implementación por etapas.
