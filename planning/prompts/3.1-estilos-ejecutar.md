# Etapa 3.1 - Ejecutar Story Spec Gamma-like

## Prompt
Implementa pipeline narrativo tipo Gamma:

1. Agregar `story_spec` como entrada de `generate_presentation`.
2. Crear conversor `story_spec -> slides` con bloques: `cover`, `section`, `insight`, `metrics`, `comparison`, `timeline`, `quote`, `cta`.
3. Exponer tool `get_story_spec_template` con schema y ejemplo.

Entrega:
- Código y funciones nuevas.
- Compatibilidad hacia atrás con `slides` directos.
