# Etapa 5.1 - Ejecutar Theme Contrast Tokens

## Prompt
Ajusta tokens globales para contraste consistente en temas no-dark:

1. Revisar variables CSS (`--color-gray-*`, foreground/background/border).
2. Asegurar contraste AA en controles base (select, input, button).
3. Evitar overrides que dejen texto gris sobre fondo gris.

Entrega:
- Cambios en CSS/tokens.
- Mapa de tokens por tema.
