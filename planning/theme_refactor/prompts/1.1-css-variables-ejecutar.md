# Etapa 1.1 - Agregar Variables CSS (Chatbox & Titulo)

## Contexto

La aplicacion tiene problemas de colores hardcodeados que solo funcionan en dark mode. Necesitamos agregar variables CSS para:

- **Chatbox**: fondo, texto, borde, ring, sombra
- **Titulo gradient**: colores de inicio y fin del gradiente

## Archivo a Modificar

```
src/tailwind.css
```

## Instrucciones

### 1. Abrir el archivo `src/tailwind.css`

### 2. Localizar el final del bloque de variables de tema existente

Buscar el comentario `/* Semantic Colors */` y la linea `--color-info: 59 130 246;` que esta aproximadamente en la linea 195-200.

### 3. Agregar DESPUES del cierre del bloque `:root` de Semantic Colors

Insertar el siguiente bloque CSS **DENTRO del mismo `@layer base`** existente, justo antes del cierre `}` del `@layer base`:

```css
/* ============================================
     CHATBOX & TITLE COLORS - Theme-aware styling
     ============================================ */

/* Chatbox & Title - Light Theme */
:root,
[data-theme='light'] {
	/* Chatbox */
	--color-chatbox-bg: 255 255 255;
	--color-chatbox-text: 23 23 23;
	--color-chatbox-border: 229 231 235;
	--color-chatbox-ring: 59 130 246 / 0.2;
	--color-chatbox-shadow: 0 4px 20px -2px rgba(0, 0, 0, 0.1);
	/* Title Gradient */
	--color-title-gradient-start: 55 65 81;
	--color-title-gradient-end: 107 114 128;
}

/* Chatbox & Title - Dark Theme */
.dark,
[data-theme='dark'] {
	/* Chatbox */
	--color-chatbox-bg: 10 10 10 / 0.8;
	--color-chatbox-text: 243 244 246;
	--color-chatbox-border: 255 255 255 / 0.05;
	--color-chatbox-ring: 255 255 255 / 0.1;
	--color-chatbox-shadow: 0 10px 40px -5px rgba(0, 0, 0, 0.5);
	/* Title Gradient */
	--color-title-gradient-start: 236 236 236;
	--color-title-gradient-end: 155 155 155;
}

/* Chatbox & Title - OLED Dark Theme */
[data-theme='oled-dark'] {
	/* Chatbox */
	--color-chatbox-bg: 0 0 0 / 0.9;
	--color-chatbox-text: 248 250 252;
	--color-chatbox-border: 255 255 255 / 0.05;
	--color-chatbox-ring: 255 255 255 / 0.1;
	--color-chatbox-shadow: 0 10px 40px -5px rgba(0, 0, 0, 0.8);
	/* Title Gradient */
	--color-title-gradient-start: 236 236 236;
	--color-title-gradient-end: 155 155 155;
}

/* Chatbox & Title - Her Theme */
[data-theme='her'] {
	/* Chatbox */
	--color-chatbox-bg: 255 250 250;
	--color-chatbox-text: 23 23 23;
	--color-chatbox-border: 254 205 211;
	--color-chatbox-ring: 244 63 94 / 0.2;
	--color-chatbox-shadow: 0 4px 20px -2px rgba(244, 63, 94, 0.1);
	/* Title Gradient */
	--color-title-gradient-start: 136 19 55;
	--color-title-gradient-end: 190 18 60;
}
```

### 4. Guardar el archivo

### 5. Verificar compilacion

El servidor de desarrollo deberia recompilar automaticamente. Si hay errores, verificar:

- Sintaxis CSS correcta (punto y coma al final de cada linea)
- Bloques cerrados correctamente
- No hay caracteres extra

## Resultado Esperado

El archivo `src/tailwind.css` deberia tener aproximadamente 50 lineas adicionales con las variables de chatbox y titulo para los 4 temas.

git checkout HEAD -- src/tailwind.css
