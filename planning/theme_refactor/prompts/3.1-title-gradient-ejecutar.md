# Etapa 3.1 - Actualizar Gradiente de Titulo

## Contexto

El titulo de la aplicacion ("Cognitia") usa un gradiente con colores hardcodeados:
- `rgb(236, 236, 236)` - gris muy claro
- `rgba(155, 155, 155, 0.8)` - gris medio

Estos colores son invisibles en light mode (texto claro sobre fondo claro).

## Archivos a Modificar

```
src/lib/components/chat/Placeholder.svelte
src/lib/components/chat/ChatPlaceholder.svelte
```

## Instrucciones

### 1. Modificar `src/lib/components/chat/Placeholder.svelte`

Buscar (aproximadamente linea 123-127):
```svelte
<h1 class="text-6xl md:text-7xl font-secondary font-light tracking-tight pb-2"
    style="background: linear-gradient(to bottom, rgb(236, 236, 236), rgba(155, 155, 155, 0.8)); -webkit-background-clip: text; background-clip: text; color: transparent;">
  {APP_NAME}
</h1>
```

Reemplazar con:
```svelte
<h1 class="text-6xl md:text-7xl font-secondary font-light tracking-tight pb-2"
    style="background: linear-gradient(to bottom, rgb(var(--color-title-gradient-start)), rgb(var(--color-title-gradient-end) / 0.8)); -webkit-background-clip: text; background-clip: text; color: transparent;">
  {APP_NAME}
</h1>
```

### 2. Modificar `src/lib/components/chat/ChatPlaceholder.svelte`

Buscar el mismo patron (aproximadamente linea 70-75):
```svelte
<h1 class="text-6xl md:text-7xl font-secondary font-light tracking-tight pb-2"
    style="background: linear-gradient(to bottom, rgb(236, 236, 236), rgba(155, 155, 155, 0.8)); -webkit-background-clip: text; background-clip: text; color: transparent;">
```

Reemplazar con el mismo codigo:
```svelte
<h1 class="text-6xl md:text-7xl font-secondary font-light tracking-tight pb-2"
    style="background: linear-gradient(to bottom, rgb(var(--color-title-gradient-start)), rgb(var(--color-title-gradient-end) / 0.8)); -webkit-background-clip: text; background-clip: text; color: transparent;">
```

### 3. Guardar ambos archivos

### 4. Verificar en el navegador

El titulo "Cognitia" deberia ser visible en todos los temas:
- **Light mode**: Gris oscuro (gray-700 a gray-500)
- **Dark mode**: Gris claro (como antes)
- **Her theme**: Tonos rose

## Detalles del Cambio

| Aspecto | Antes | Despues |
|---------|-------|---------|
| Color inicio | `rgb(236, 236, 236)` | `rgb(var(--color-title-gradient-start))` |
| Color fin | `rgba(155, 155, 155, 0.8)` | `rgb(var(--color-title-gradient-end) / 0.8)` |
| Theme-aware | No | Si |

## Nota Tecnica

La sintaxis `rgb(var(--variable) / 0.8)` es la forma correcta en CSS moderno para aplicar alpha a un valor RGB de variable CSS. Esto funciona porque nuestras variables usan el formato `R G B` (space-separated).
