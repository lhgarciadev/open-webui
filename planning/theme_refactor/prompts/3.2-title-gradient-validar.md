# Etapa 3.2 - Validar Gradiente de Titulo

## Objetivo

Verificar que el titulo de la aplicacion es visible en todos los temas.

## Pre-requisitos

- Etapa 3.1 completada
- Servidor de desarrollo corriendo
- Navegador en la pagina principal (donde se muestra el placeholder)

## Validacion por DevTools

Ejecutar en la consola:

```javascript
// Verificar gradiente del titulo
(function testTitleGradient() {
  // Buscar el h1 del titulo
  const titles = document.querySelectorAll('h1');
  let titleElement = null;

  titles.forEach(h1 => {
    if (h1.style.backgroundClip === 'text' || h1.style.webkitBackgroundClip === 'text') {
      titleElement = h1;
    }
  });

  if (!titleElement) {
    console.log('⚠️ Titulo con gradiente no encontrado en esta pagina');
    console.log('Navegar a un chat nuevo para ver el placeholder');
    return;
  }

  const style = titleElement.getAttribute('style');
  const usesVariable = style.includes('var(--color-title-gradient');

  if (usesVariable) {
    console.log('✅ PASS: Titulo usa variables CSS');
    console.log('Style:', style);
  } else {
    console.log('❌ FAIL: Titulo aun usa colores hardcoded');
    console.log('Style:', style);
  }

  // Verificar valores de las variables
  const root = document.documentElement;
  const start = getComputedStyle(root).getPropertyValue('--color-title-gradient-start').trim();
  const end = getComputedStyle(root).getPropertyValue('--color-title-gradient-end').trim();

  console.log('Gradient start:', start);
  console.log('Gradient end:', end);
})();
```

## Criterios de Exito

- [ ] Light theme: Titulo gris oscuro, claramente legible
- [ ] Dark theme: Titulo gris claro (comportamiento original)
- [ ] OLED theme: Titulo visible sobre fondo negro
- [ ] Her theme: Titulo con tonos rose
- [ ] No hay colores RGB hardcodeados en el style
- [ ] Ambos archivos (Placeholder.svelte y ChatPlaceholder.svelte) actualizados

## Problemas Comunes

### Titulo invisible en light mode
- Verificar que las variables de etapa 1 se compilaron correctamente
- Verificar que el tema light tiene `--color-title-gradient-start: 55 65 81`

### Gradiente no se aplica
- Verificar sintaxis: `rgb(var(--variable) / 0.8)` (no `rgba`)
- Verificar que `-webkit-background-clip: text` esta presente

### Solo un archivo actualizado
- Verificar que AMBOS archivos fueron modificados:
  - `Placeholder.svelte`
  - `ChatPlaceholder.svelte`
